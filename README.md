## 这是个啥?

React+Redux 后台管理系统脚手架<br/> React Hooks / Redux / React Router / antd / webpack4 / Javascript

## 构建 Start

```javascript
npm install       // 安装依赖模块
npm start         // 运行开发环境，默认监听8888端口
npm run build         // 正式打包，生成最终代码
npm run dist          // 运行正式打包后的最终代码，默认监听8888端口
```
